# File-Packer-Unpacker-Java
Java project for packing and unpacking files with encryption using Swing GUI.

# File Packer & Unpacker with Encryption

## Technology Used
- Java
- Swing (GUI)
- File I/O
- Encryption & Decryption

## Features
- Pack multiple files
- Preserve metadata
- Encrypt packed file
- GUI Support

## How to Run

### Packing
java FilePacker Demo Pack.txt

### Unpacking
java FileUnpacker Pack.txt
